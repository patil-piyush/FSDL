# Assignment 4 – FinFlow Expense Tracker Dashboard

## Objective

To develop a modern **Expense Tracker Dashboard** that allows users to manage income and expenses, visualize financial data using charts, and maintain transaction history through an interactive and responsive interface.

## How to Run

1. Open the `Assignment-4` folder.
2. Launch `index.html` in a modern web browser.
3. Navigate to **Add New** to record transactions.
4. View analytics on the dashboard.
5. Browse all transactions in the **History** page.
6. Switch between Dark and Light themes.
7. Print transaction history if required.
