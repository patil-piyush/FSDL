# Project 1 – Dynamic Login System

## Overview

A simple authentication system that validates user credentials and redirects authenticated users to a protected dashboard.

### Features

* User login page
* Client-side authentication
* Dashboard protection
* Logout functionality
* Session management using Local Storage
* Responsive dark-themed UI
* Product showcase dashboard

# Project 2 – E-Commerce Website

## Overview

* Product Listing
* Search Products
* Price Filtering
* Add to Cart

## How to Run

### Dynamic Login System

1. Open the `Dynamic Login Page` folder.
2. Launch `login.html` in a web browser.
3. Login using the provided credentials.
4. Explore the dashboard and logout functionality.

### E-Commerce Website

1. Open the `Ecommerce` folder.
2. Launch `index.html`.
3. Register a new account.
4. Login with the registered credentials.
5. Browse products.
6. Search or filter products.
7. Add items to the cart.
8. Proceed to checkout.
9. View profile and order history.
