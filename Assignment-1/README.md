# Assignment 1 – HTML Text Formatting Tags

## Objective

To demonstrate the use of commonly used **HTML text formatting tags** by creating:

* A basic HTML page using only HTML.
* A styled version of the same page using CSS.

---

## Files

```text
Assignment-1/
├── FormattingTags.html
└── FormattingTagsWithCSS.html
```

### 1. `FormattingTags.html`

A basic HTML page that demonstrates various HTML text formatting tags using a table. No CSS styling is applied.

### 2. `FormattingTagsWithCSS.html`

An enhanced version of the same page with CSS styling, including:

* Modern layout
* Responsive container
* Styled table
* Improved typography
* Highlighted code blocks
* Better readability and user experience

---

## HTML Tags Demonstrated

| Tag            | Purpose                               |
| -------------- | ------------------------------------- |
| `<b>`          | Bold text                             |
| `<strong>`     | Important (semantic bold) text        |
| `<i>`          | Italic text                           |
| `<em>`         | Emphasized text                       |
| `<u>`          | Underlined text                       |
| `<mark>`       | Highlighted text                      |
| `<small>`      | Smaller text                          |
| `<del>`        | Deleted text                          |
| `<ins>`        | Inserted text                         |
| `<sub>`        | Subscript text                        |
| `<sup>`        | Superscript text                      |
| `<code>`       | Inline code                           |
| `<pre>`        | Preformatted text                     |
| `<kbd>`        | Keyboard input                        |
| `<samp>`       | Sample output *(HTML version only)*   |
| `<var>`        | Variable *(HTML version only)*        |
| `<abbr>`       | Abbreviation                          |
| `<cite>`       | Title of a work *(HTML version only)* |
| `<q>`          | Short quotation                       |
| `<blockquote>` | Block quotation                       |

---

## Features

### FormattingTags.html

* Pure HTML implementation
* Uses a table to display formatting tags
* Simple and easy to understand
* Suitable for beginners

### FormattingTagsWithCSS.html

* Modern and clean UI
* Styled table with hover effects
* Responsive centered layout
* Improved typography
* Styled `<code>`, `<pre>`, `<kbd>`, and `<blockquote>` elements

---

## How to Run

1. Open the `Assignment-1` folder.
2. Double-click either HTML file, or open it in any modern web browser.
3. Compare the basic HTML version with the CSS-styled version.

---

## Learning Outcomes

After completing this assignment, you will understand:

* Basic HTML page structure
* Common HTML text formatting tags
* Semantic vs. non-semantic formatting tags
* Creating tables in HTML
* Applying CSS to improve the appearance of a webpage
